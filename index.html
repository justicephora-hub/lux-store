Deploy to GitHub Pages

1
Download your website files

You need: index.html, admin.html, client-portal.html, and setup-admin.html

2
Create a GitHub repository

Go to github.com → New Repository → name it clearpath-website → set to Public

3
Upload your files

Click "Upload files" in your repo, drag all HTML files in, then click "Commit changes"

4
Enable GitHub Pages

Go to Settings → Pages → Source: Deploy from branch → Branch: main → Save

Settings → Pages → Branch: main → Save
5
Your site goes live at:
https://yourusername.github.io/clearpath-website/
6
Access your admin dashboard at:
https://yourusername.github.io/clearpath-website/admin.html
Important: Your credentials are saved in your browser's localStorage. When you open the admin page on GitHub Pages for the first time on a new device, run this setup page once to recreate them.

Delete or hide setup-admin.html from your public GitHub repo after setup — or password-protect it. Anyone can access it otherwise.
